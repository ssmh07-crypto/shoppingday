# 네이버 검색 품질 가이드 적용 기준

마지막 확인: 2026-07-19

이 문서는 네이버 검색어드바이저 공식 가이드를 Shoppingday의 키워드 후보와 상품명
초안에 일관되게 적용하기 위한 내부 기준이다. 키워드·상품명·검색 노출 품질을 변경할
때는 `docs/product-strategy.md`를 먼저 읽고 이 문서를 함께 참고한다.

## 적용 범위와 주의점

네이버 검색어드바이저는 기본적으로 **네이버 웹 검색과 웹사이트 SEO**를 설명한다.
스마트스토어 쇼핑 검색의 노출 순위 공식이나 상품명 작성 규정 자체가 아니다.
따라서 아래 원칙은 상품명 품질을 높이는 참고 기준으로 사용하되, 특정 키워드나 제목이
쇼핑 검색 순위를 올린다고 보장하거나 실제 순위·검색량처럼 표시하지 않는다.

- 웹문서 제목 40자 권장은 품질 경고 기준으로만 사용한다. 상품명 저장을 막는 강제
  제한으로 쓰지 않는다.
- 구조화 데이터, robots.txt, 사이트맵, canonical은 자체 웹사이트를 운영할 때 검토할
  항목이다. 현재 스마트스토어 상품명 생성 로직에는 적용하지 않는다.
- 구조화 데이터 적용도 검색 노출을 보장하지 않는다.
- 실제 검색량은 네이버 검색광고 API 응답만 사실 데이터로 표시한다.

직접 참고하는 일반 품질 원칙은 사용자 중심 정보, 정확하고 고유한 제목, 불필요한
키워드 남용과 중복 표현 방지, 콘텐츠 주제와 제목의 일치, 상위 노출을 보장하지 않는
표현이다.

다음 웹사이트 SEO 항목은 스마트스토어 상품 키워드 추천 규칙에 직접 적용하지 않는다:
`robots.txt`, 사이트맵, HTML `title` 태그, `meta description`, Open Graph, 웹문서 수집·색인,
구조화 데이터, 웹사이트 URL 구조, 네이버 웹마스터도구 리포트.

## Shoppingday에 적용할 원칙

### 1. 정확하고 간결한 상품명

- 제목만 보고 실제 상품을 구분할 수 있어야 한다.
- 구체적인 상품 유형은 유지한다. 예: `골무`.
- 같은 단어를 반복하지 않는다.
- 구체적인 상품 유형이 있으면 `부자재`, `용품`, `상품`, `제품`, `세트`, `도구`처럼
  넓은 분류어는 초안에서 우선 제외한다.
- 40자를 넘으면 간결하게 다듬으라는 안내를 표시하되 저장은 허용한다.

### 2. 관련 없는 인기 검색어와 모순 속성 차단

- 검색량이 높아도 상품과 직접 관련 없는 단어는 후보나 상품명에 넣지 않는다.
- 소재, 색상, 대상처럼 상품 입력값과 충돌하는 연관 검색어는 제외한다.
- 검색량이 많다는 이유만으로 실제 상품명을 다른 인기 품목명으로 바꾸지 않는다.

### 3. 홍보성·스팸성 표현 제외

`무료배송`, `정품`, `신상품`, `인기`, `추천`, `특가`, `할인`, `이벤트`, `국내배송`,
`당일배송` 같은 홍보 문구는 규칙 기반 상품명 초안에서 제외한다. 상품의 객관적인
속성보다 클릭을 유도하는 문구를 우선하지 않는다.

### 4. 사람이 최종 결정

- 생성 결과는 수정 가능한 초안으로 제공한다.
- 품질 문제는 경고하되 단순 권장사항만으로 저장을 막지 않는다.
- 기존에 저장한 상품명을 자동으로 덮어쓰지 않는다.
- 상품명 변경 이력과 변경 후 실제 반응 비교는 성장 관리 단계에서 다룬다.

### 5. 1,000개 상품 슬롯 관점

- 빠른 등록 단계에는 규칙 기반 기본 모드로 중복·일반 분류어·홍보어를 제거한다.
- 고비용 분석은 실제 판매 반응이 확인된 상품에만 선택적으로 사용한다.
- 동일·유사 상품명 경고로 제한된 슬롯을 서로 구분되는 상품으로 운영한다.

## 현재 구현 규칙

상품명 초안은 다음 순서로 만든다.

1. 선택 키워드를 낱말 단위로 정리하고 동일 단어를 한 번만 남긴다.
2. 사용자가 입력한 금지어와 홍보성 표현을 제거한다.
3. 구체적인 상품 유형이 있으면 넓은 분류어를 제거한다.
4. 기본은 소재·용도 같은 속성을 먼저 배치하되, 부품·소모품 계열처럼 품목을 먼저
   읽는 편이 자연스러운 카테고리는 별도 템플릿을 사용한다.
5. 편집 화면에서 상품 유형 누락, 단어 반복, 넓은 분류어, 홍보어, 40자 초과를
   실시간으로 안내한다.

상품 관리의 판매용 상품명 추천도 같은 원칙을 사용한다. 현재 판매용 상품명과 선택된
네이버 카테고리에서 상품 유형·소재·재질·용도를 규칙으로 분리하고, 네이버 검색광고
키워드 도구는 관련 조합의 실제 월간 검색량을 확인하는 용도로만 사용한다. 연관
키워드가 상품 유형을 포함하지 않거나 입력 소재와 충돌하면 추천 근거에서 제외한다.
추천 결과는 미리보기 후 사용자가 적용하며 자동 저장하거나 네이버 상품에 자동
반영하지 않는다.

이 규칙은 검색순위 상승을 보장하는 최적화 공식이 아니라, 부정확하거나 반복적인
상품명을 줄이기 위한 품질 안전장치다.
